# 相机激光雷达联合标定

## 📄 1️⃣ image_click.py（图像点选）
👉 你点 6–10 个角点即可。
> 注意：图片的点选择顺序必须和点云完全一致。    
> 可尽量多选择一些点，推荐6-10个角点。    
> 一种方式：从左上角开始，顺时针点3圈。

## 📄 2️⃣ cloud_click.py（点云点选）
📌 不要求点完全准确，只要“几何一致”
> 注意：点云的点选择顺序必须和图像完全一致。

## 📄 3️⃣ optimize_extrinsic.py（标定核心）
👉 运行该文件，会自动生成标定结果。
> 注意：请先运行 image_click.py 和 cloud_click.py    
> 运行该脚本前，需要对相机提前进行标定，此时需要用到网格标定板。

## 📄 4️⃣ validate_projection.py（结果验证）
👉 这一步你会非常直观地知道标定有没有成功。
